		"""SCPI: MMEMory:STORe<n>:SPURious \n

		Exports the marker peak list available for spurious emission measurements to a file. Secure User Mode In secure user mode,
		settings that are stored on the instrument are stored to volatile memory, which is restricted to 256 MB. Thus, a 'memory
