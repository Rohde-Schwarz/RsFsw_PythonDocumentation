# SCPI: TRIGger[:SEQuence]:LEVel:IQPower
value: float = driver.applications.k91Wlan.trigger.sequence.level.iqPower.get()

This command defines the magnitude the I/Q data must exceed to cause a trigger event. Note that any RF attenuation or preamplification is considered when the trigger level is analyzed. If defined, a reference level offset is also considered. For details on the trigger source see ‘Trigger Source’.
