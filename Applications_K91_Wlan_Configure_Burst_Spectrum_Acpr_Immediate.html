Immediate

SCPI Command :

CONFigure:BURSt:SPECtrum:ACPR[:IMMediate]
class ImmediateCls[source]

Immediate commands group definition. 1 total commands, 0 Subgroups, 1 group commands

set(opc_timeout_ms: int = - 1) → None[source]
# SCPI: CONFigure:BURSt:SPECtrum:ACPR[:IMMediate]
driver.applications.k91Wlan.configure.burst.spectrum.acpr.immediate.set()

This remote control command configures the result display in window 2 to be ‘ACPR’ (adjacent channel power relative) . Results are only displayed after a measurement is executed, e.g. using the method RsFsw.Applications.K10x_Lte.Initiate. Immediate.set command. Note that the CONF:BURS:<ResultType>:IMM commands change the screen layout to display the ‘Magnitude Capture’ buffer in window 1 at the top of the screen and the selected result type in window 2 below that. Any other active windows are closed. Use the LAYout commands to change the display (see ‘Configuring the result display’) .

param opc_timeout_ms

Maximum time to wait in milliseconds, valid only for this call.
